# Smart News Explorer

## Project Description

Smart News Explorer is a web application that allows users to explore real-time news across multiple categories such as Technology, Business, Sports, and Health. The application is designed to provide a clean and interactive interface where users can search for news articles, filter them by category, and sort them based on recency.

By default, the application focuses on technology news to maintain a clean and relevant starting point, while also allowing users to dynamically switch between different categories for a broader experience.

## API Used

News API (from Public API Lists)

The application uses a public news API to fetch real-time news data including headlines, descriptions, images, and publication dates. The API supports category-based filtering and keyword-based search.

## Features

* View latest news articles (default: Technology)
* Search news using keywords (e.g., AI, Startups, Web Development)
* Filter news by categories:

  * Technology
  * Business
  * Sports
  * Health
* Sort articles by latest or oldest
* Responsive design for mobile, tablet, and desktop

## Planned Enhancements

* Save favorite articles using localStorage
* Dark mode / Light mode toggle

## Technologies Used

* HTML
* CSS
* JavaScript
* Fetch API
* Array Higher Order Functions (map, filter, sort)

## How to Run

1. Clone the repository
2. Open the project folder
3. Open index.html in your browser
4. Add your API key in the JavaScript file
